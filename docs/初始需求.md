# 易拍

英文：easy-photo

产品功能描述：该应用的主要功能是给用户拍照教程

## 管理端：

一、分类管理

管理员可以对下列分类进行管理，分类分为两级

一级分类：

1.风格分类，下属二级分类：例如生活风，甜美风、忧郁风，自然风等

2.姿势分类，下属二级分类：例如站立，坐姿，半身，蹲姿等

3.色调分类，下属二级分类：例如冷暖色调，绿色、蓝色等

二、用户管理

可以新增或删除用户

### 用户端：

分类以tab展示，选中一级分类，可以继续选择二级分类，选中二级分类后展示该分类下的教程卡片列表，卡片具有封面和标题

教程包括文字教程，图片示例，图片可以包含多张

教程中的文字分点展示，有滤镜、调色、修图建议

用户可以浏览其他用户上传的教程，也可以查看其他用户上传的教程

可以设置教程的可见范围，可选择public和private，public所有用户可见，private则只有自己可见

管理员可以看见所有的教程

用户可以收藏、点赞某一教程

收藏页显示用户收藏的教程

## 技术栈

前端使用vue3+elementUI

页面主色调为淡黄色，可以加入莫兰迪色系的、淡黄色或米白色

后端使用java8、springboot、Mysql
